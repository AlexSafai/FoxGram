# FoxGram

Minimal Lit-based Instagram-style feed demo.

Quick start:

1. Install a static server (if you don't have one):

```bash
npm install -g http-server
```

2. Run the dev server from the project root:

```bash
npm start
```

3. Open http://localhost:3000

Vercel API:

- The API function is at `/api/get-posts.js` and reads `data.json`.
